# Final Project GUI
Turn Based fight between the player and one monster!
## Final Project Description
_This boss fight game will be between the player and the monster, in which the player has to be lucky enough to race against 
the monster trading blows to deplete the other's health based on random numbers dealing damage, where the player has better odds._
